## 1. 产品概述

校园二手交易平台——面向高校学生的闲置物品流转平台，解决毕业季/换季时大量闲置物品无处处置的痛点。用户可快速发布二手商品、按分类筛选浏览、与卖家直接联系完成交易。

- 目标用户：在校大学生群体，交易场景集中在校园内
- 核心价值：零手续费、同学间高信任、当面交易免物流

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 校园邮箱注册 | 浏览、搜索、发布、收藏、联系卖家 |

### 2.2 功能模块

1. **首页**：分类导航栏、搜索框、商品卡片列表、分类筛选标签
2. **商品详情页**：图片轮播、商品信息、卖家信息与联系方式、收藏按钮
3. **发布页**：商品表单（标题/描述/价格/分类/成色）、图片上传、发布操作
4. **个人中心**：我的发布列表、我的收藏列表、用户信息展示

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | 搜索栏 | 支持关键词搜索商品，实时过滤 |
| 首页 | 分类标签栏 | 横向滚动分类标签，点击筛选对应分类商品 |
| 首页 | 商品列表 | 卡片式展示商品图片、标题、价格、成色，支持下拉加载 |
| 商品详情页 | 图片轮播 | 支持左右滑动/点击切换商品图片，含指示器 |
| 商品详情页 | 商品信息 | 标题、价格、成色、描述、发布时间 |
| 商品详情页 | 卖家信息 | 卖家头像、昵称、联系方式（微信/QQ/手机） |
| 商品详情页 | 操作栏 | 收藏按钮、联系卖家按钮 |
| 发布页 | 商品表单 | 标题输入、描述多行输入、价格输入、分类下拉、成色选择 |
| 发布页 | 图片上传 | 支持多图上传（最多6张），预览与删除 |
| 发布页 | 发布按钮 | 校验必填项后提交发布 |
| 个人中心 | 用户信息 | 头像、昵称、校园认证标识 |
| 个人中心 | 我的发布 | 已发布商品列表，支持删除/编辑状态 |
| 个人中心 | 我的收藏 | 收藏商品列表，支持取消收藏 |

## 3. 核心流程

用户浏览与购买流程：用户进入首页 → 浏览商品列表/使用分类筛选/搜索 → 点击商品卡片进入详情页 → 查看图片与描述 → 点击联系卖家 → 完成线下交易。

用户发布流程：用户点击发布入口 → 填写商品信息 → 上传图片 → 选择分类与成色 → 设定价格 → 提交发布 → 商品出现在首页列表。

## 4. 用户界面设计

### 4.1 设计风格

- 主色：暖橙 `#F97316`，辅色：浅橙 `#FFF7ED`，强调色：深橙 `#EA580C`
- 背景：纯白 `#FFFFFF`，卡片：白底 + 浅灰边框 + 柔和阴影
- 按钮：圆角（rounded-xl），主按钮橙色实心，次按钮橙色描边
- 字体：标题用粗体大号，正文用常规体，辅助信息用小号灰色字
- 布局：卡片式网格布局，移动端单列，桌面端多列自适应
- 图标：线性风格，与暖橙色调一致

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | 搜索栏 | 圆角输入框 + 搜索图标，橙色聚焦边框 |
| 首页 | 分类标签 | 横向滚动胶囊标签，选中态橙色填充 |
| 首页 | 商品卡片 | 圆角卡片，图片在上、信息在下，价格橙色高亮 |
| 商品详情页 | 图片轮播 | 大图区域 + 底部圆点指示器，支持手势滑动 |
| 商品详情页 | 信息区 | 标题粗体、价格大号橙色、成色标签、描述文字 |
| 商品详情页 | 卖家卡片 | 头像 + 昵称 + 联系方式按钮组 |
| 商品详情页 | 底部操作栏 | 固定底栏：收藏图标 + 联系卖家按钮 |
| 发布页 | 表单 | 输入框橙色聚焦态，分类胶囊选择器，成色滑块 |
| 发布页 | 图片上传 | 九宫格预览 + 加号添加按钮，支持拖拽排序 |
| 个人中心 | 用户卡片 | 顶部橙色渐变背景 + 头像居中 + 昵称 |
| 个人中心 | 标签切换 | 我的发布 / 我的收藏 标签页切换 |
| 个人中心 | 商品列表 | 复用首页商品卡片，支持编辑/删除操作 |

### 4.3 响应式设计

- 桌面优先，移动端自适应
- 商品卡片网格：桌面4列、平板2列、手机1列
- 底部导航栏：移动端固定底栏，桌面端顶部导航
- 触摸优化：按钮最小 44px 点击区域，卡片点击反馈
